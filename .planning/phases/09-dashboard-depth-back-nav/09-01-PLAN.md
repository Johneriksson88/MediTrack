<objective>
Slice A — ORD-10 back-nav fix.

Implement a single source of truth back-link helper hook (`useBestallningarBackLink`) and propagate `?from=<status>` at all four existing navigation entry points (BestallningarPage's Ny beställning + Utkast row clicks; OrdersTable + OrdersCardList row clicks for the non-Utkast tabs). Rewire the four `<Link to="/bestallningar">` sites + the one `navigate('/bestallningar')` in `ComposeOrderPage.tsx` to consume the hook. Honor an `order.status` fallback when `?from=` is missing.

Purpose: SC#3 says "Tillbaka till beställningar returns the user to the previously-active status tab, not the default Utkast tab" — today every back-link goes to bare `/bestallningar` (which defaults to Utkast). SC#4 says the fix must work from tab clicks, deep links, AND the new dashboard card. This slice covers the first two (tab clicks + deep links + Ny beställning) and lays the contract that Slice C consumes for the dashboard card row links.

Output: One new helper hook + one new hook test file + edits to 4 existing route/component files + 2 extended test files. Demoable end-to-end on its own (independent of Slice B and C).
</objective>
