## STRIDE Threat Register (ASVS L1)

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-06 | Tampering | `?class=` query param could carry malicious payload | mitigate | `medicationListQuery.therapeuticClass = therapeuticClassEnum.optional()` rejects non-enum values at the Zod boundary; Fastify returns 400 before service runs. | T-06-08 | Information Disclosure | Audit allowlist extension includes `therapeuticClass` in `before`/`after` JSON of Medication update events | accept | `therapeuticClass` is non-sensitive metadata (clinical classification, not PII). | T-06-09 | Denial of Service | New `@@index([therapeuticClass])` index adds write overhead on every Medication UPDATE | accept | Single-column index on a low-cardinality enum (14 values). Write overhead is negligible vs SELECT speedup for the filter combobox. Index size on 43,538 rows is < 1 MB. |
