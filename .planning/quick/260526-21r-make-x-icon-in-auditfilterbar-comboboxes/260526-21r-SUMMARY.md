# Quick 260526-21r: Make X icon in AuditFilterBar comboboxes clickable per-filter clear

One-liner: Replace decorative `<X />` inside three Radix PopoverTrigger Buttons (Användare / Entitetstyp / Åtgärd) with focusable `role="button"` spans that intercept pointerdown to clear ONLY that filter without opening the popover.

## What changed

Three structurally identical sibling edits inside `apps/web/src/routes/admin/AuditFilterBar.tsx` — one per combobox. Each replaces this pattern:

```tsx
<X className="h-3 w-3 ml-2 flex-shrink-0" aria-hidden="true" />
```

with:

```tsx
<span
  role="button"
  tabIndex={0}
  aria-label="Rensa <användarfilter | entitetstypsfilter | åtgärdsfilter>"
  onPointerDown={(e) => {
    e.stopPropagation();
    e.preventDefault();
    onChange({ <actor | entity | action>: '' });
  }}
  onClick={(e) => {
    e.stopPropagation();
  }}
  onKeyDown={(e) => {
    if (e.key === 'Enter' || e.key === ' ') {
      e.stopPropagation();
      e.preventDefault();
      onChange({ <actor | entity | action>: '' });
    }
  }}
  className="ml-2 flex-shrink-0 inline-flex h-4 w-4 items-center justify-center
             rounded-sm hover:bg-foreground/10 focus-visible:outline-none
             focus-visible:ring-2 focus-visible:ring-primary cursor-pointer"
>
  <X className="h-3 w-3" aria-hidden="true" />
</span>
```

| Combobox     | aria-label                | onChange patch        | Line (pre-edit) |
| ------------ | ------------------------- | --------------------- | --------------- |
| Användare    | Rensa användarfilter      | `{ actor: '' }`       | ~152            |
| Entitetstyp  | Rensa entitetstypsfilter  | `{ entity: '' }`      | ~214            |
| Åtgärd       | Rensa åtgärdsfilter       | `{ action: '' }`      | ~270            |

The `▾` caret hint else-branch is untouched, as are the global "Rensa filter" button (lines 311-321 → now 377-387) and the requestId chip dismiss button (lines 333-342 → now 399-408).
